# Setup

## Prereqs
- Node ≥ 18.18
- npm ≥ 8

## Install & Run
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Env vars (.env.local)
```
NEXT_PUBLIC_MODE_DEFAULT=recruiter
NEXT_PUBLIC_ASSETS_BASE=https://your-cdn-endpoint.com
RESEND_API_KEY=your_resend_api_key_here
NEXT_PUBLIC_SITE_URL=https://yourdomain.com
```
